SOBRE O PROJETO

O projeto faz parte da Unidade Escolar ETEC Prof. Maria Cristina Medeiros, acontece no 1º Ano do Ensino Médio – Integrado ao Técnico em Informática para Internet, período manhã.

Este é um projeto Interdisciplinar entre os componentes Artes, Literatura e Informática, este site trata diferentes assuntos, como:
  • Revista Tudo.Cultura
  • Gêneros Literários e Textuais
  • Trovadorismo
  • Arte Mexicana

Desenvolvido de Junho a Setembro de 2020. O objetivo é unir diferentes componentes de forma diversificada e já se preparando para o vestibular. O Intuito é além de tudo trabalhar algumas competências socioemocionais como: exercitar a curiosidade, criatividade, aprender a trabalhar em equipe e ao mesmo tempo a autonomia e responsabilidade das entregas semanais do projeto.

Disciplinas e Orientadores

INFORMÁTICA PARA WEB

Neste componente aprendemos a criar e editar sites com HTML e CSS que se adaptam a diferentes telas. Curso de Informática para Internet da ETEC MCM.

Orientador responsável pela disciplina: Prof. Cíntia Pinho.
PORTUGUÊS E LITERATURA

Este é um componente da base comum do Ensino Médio, no qual estamos fazendo um trabalho sobre literatura que engloba o Período do Trovadorismo.

Orientador responsável pela disciplina: Prof. Bárbara Tomazia.
ARTES

Este é um componente da base comum do Ensino Médio, no qual estamos fazendo um trabalho sobre Arte Mexicana, o qual podemos trabalhar com Teatro, Música ou Quadros.

Linguagens e Interfaces

Veja abaixo as linguagens e interfaces de programação que foram usadas para desenvolver o site da revista interdisciplinar Tudo.Cultura:

HTML

Linguagem de programação

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Bootstrap

Framework front-end

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript.

CSS

Linguagem de folhas de estilo

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags style.

JavaScript

Linguagem de programação

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

jQuery

Interface de programação de aplicações

jQuery é uma biblioteca de funções JavaScript que interage com o HTML, desenvolvida para simplificar os scripts interpretados no navegador do cliente.
